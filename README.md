# RealEstatePro

Real Estate Pro is a full-stack real estate application developed using
Flutter and Laravel.

## Admin Panel (Laravel)
- Property management (Add, Edit, Delete)
- Category & City management
- Property image uploads
- User authentication
- REST API for mobile app
- Wishlist & visits management

## Mobile App (Flutter)
- Property listing
- Filter by city & category
- Property details with images
- Wishlist support
- API-based data loading

## Tech Stack
- Flutter
- Laravel
- MySQL
- REST APIs
